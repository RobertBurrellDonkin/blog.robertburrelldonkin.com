# Containers

Based on https://aws.amazon.com/containers/?nc2=h_ql_prod_ctn


| Stack                      | Public Cloud                              | On Premises              |
| -------------------------- | ------------------------------------------| ------------------------ |
| OpenContainers eg Docker   | Amazon Elastic Container Service **ECS**  | Amazon **ECS** Anywhere |
| Kubernates                 | Amazon Elastic Kubernetes Service **EKS** | Amazon **EKS** Anywhere |


| Platform                   | Public Cloud                             |
| -------------------------- | ---------------------------------------- |
| Serverless                 | AWS **Fargate**                          |
| Servers                    | Amazon Elastic Compute Cloud **EC2**     |


| Solution                           | Public Cloud     | On Premises       |
| ---------------------------------- | ---------------- | ----------------- |
| **Networking** (Application Layer) | AWS **App Mesh** |  AWS **Outposts** |


| Space                         | Solution                                  |
| ----------------------------- | ----------------------------------------- |
| Container registry            | Amazon Elastic Container Registry **ECR** |
| Kubernates distribution       | Amazon **EKS Distro**                     |
| Deployment pipelines          | AWS **Proton**                            |
| Containerization tool         | AWS **App2Container**                     |
| Command line deployment pipelines | AWS **AWS Copilot**                   |
