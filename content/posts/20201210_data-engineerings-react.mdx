In no particular order –

 * Users are important. Every engineer should know about user interfaces.
 * Data visualization is important. Be that data engineer on the team – the one able to throw an internal app together.
 * The art in data engineering is all about shaping the flow all the way from source to sink, all the way to the user.
Learn about the front end to understand the complete journey.
